sobre os comandos:
pwd: verificar em qual diretório está;
ls: verifica o que tem na pasta;
ll: mostra o que tem na pasta em lista e também os arquivos ocultos;
cd: muda de pasta;
cd ..: volta para a pasta anterior;
apt-get (install ou removve): serve para instalar ou remover arquivos;
sudo: da permissão root para usar os arquivos;
sudo su: deixa o usuário como root e não precisa ficar colocado sudo antes de executar os comandos;
mv (mover ou renomear): usado para mover arquivos ou pastas ou nenomeá-las, lembrando que se o nome possuir espaço e necessário fechá-lo em aspas duplas.

 compilando em ubuntu 14.04	
Este comando pode ser executado em qualquer lugar:
apt-get install subversion autoconf build-essential pkg-config libboost-dev libgmp3-dev libxml2-dev liblua5.1-0-dev libmysqlclient-dev libcrypto++-dev ccache libboost-filesystem-dev libboost-regex-dev libboost-system-dev libboost-thread-dev screen libssl-dev

Este deve ser executado dentro da pasta src do servidor:
chmod 777 autogen.sh
chmod 777 build.sh
./autogen.sh
./configure --enable-server-diag --enable-mysql --enable-root-permission
./build.sh
